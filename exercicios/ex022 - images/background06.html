Background Shorthand

Shorthands for background:

background: color > image > position > repeat > attachment

For this site, the background shorthand is this:

background: black url(images/wallpaper002.jpg) center top no-repeat fixed;

Centering a box inside of a box

To center a box inside of a box, you'll need to change the position of the outside box to relative(default position of a box is already relative).

Now for the box you want centered, you'll need to set the position to 'absolute' so that you can move it left, right, top and bottom. Once you do that, set the left and top to 50% so that the box aligns in the center. You'll see that the left top corner of the box is aligned in the midle of the outside box

To align correctly we'll need to use the 'transform' property like this

transform: translate(-50%, -50%);
