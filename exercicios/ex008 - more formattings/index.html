More Formattings to Learn

Computer Code

The tag <code> is used to define a piece of Computer Code. Whatever is inside this tag, will be displayed in the browser's default monospace font


Preformatted Text

The tag <pre> is used to define preformatted text. 

    Any text inside a pre tag
    is displayed exactly as it's 
    written in the HTML
    source code

Short Quotation

The <q> tag defines a short quotation, for example:

Like my friend's dad says: The internet is a really fast idiot

Long Quotations

To do longer quotations, use the tag <blockquote>, like so:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

You can put a link to your source by adding <blockquote cite"link-to-source">

Abbreviation

When abbreviating a text, use the tag <abbr title=""> so that the full name appears once your mouse is over the abbreviation
