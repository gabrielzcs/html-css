CSS - Cascading Style Sheets

CSS is used to edit the styles of your website

You can use External CSS (normally used so you can style all pages at once) by styling it in a .css file

To do that your need to link your css file to your HTML file or files. Here is how you do it:

<link rel="stylesheet" href="file-name.css">

Write that in your <head> tag and you are good to go
