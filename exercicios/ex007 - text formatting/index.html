Main Formattings

Bold / Strong

This frase has a bold term (without semantics)

To create a bold term, use the tag <b>bold term here</b>


This frase has a strong term (with semantics)

To create a strong term, use the tag <strong>strong term here</strong>

Strong tag(<strong>) is more used nowadays, the bold tag hasn't been used as much


Italic / Emphasis

This frase has an italic term (without semantics)

To create an italic term, use the tag <i>italic term here</i>


This frase has an emphasi term (with semantics)

To create an Emphasis term, use the tag <em>emphasis term here</em>

Emphasis tag(<em>) is more use nowadays, the italic tag hasn't been used as much


Big & Small texts

Your can also create Big Texts and Small Texts using the following tags:

<big> and <small>

The <big> tag has been disuse


Deleted Text

You can show a text has been deleted by using the following command:

<del>your deleted text here</del>


Inserted Text

When you insert a text into a document, you can use the tag:

<ins>inserted text here</ins> (with semantics)

And the wrong way to do it, is by using the tag <u> (without semantics)


Superscript and Subscript Text

To to a Superscript text like so: x20+3
you can use the tag <sup>: x<sup>20</sup>+3

And to do a Subscript text like so: H2O
you can use the tag <sub>: H<sub>2</sub>O

Marked Text

This frase has a marked text

To create a marked text, use the tag <mark>marked text here</mark>


To add a CSS style tag on all tags of the same family, you can write the following command to your <head> tag
<style>
mark {
background-color: your-color;
}
</style>


Or put it directly into the tag you want to change (but it will only alter that tag):

<mark style="background-color: lime;">Like This</mark>
