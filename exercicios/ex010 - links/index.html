Links

Using External Links

To create an external link, use the tag: <a href="link-here">Name of your link </a>

Example of an external link:

My YouTube channel

To make so your page doesn't close, use the attribute: target="_blank", and rel="external"
to indicate that it's an external link you're using

My Twitch channel

Using Internal Links

When using Internal Links, you'll need to add the attributes: target="next, prev", and rel="self, nofollow"

To create an Internal Link, you need to a have another HTML file in your folder. And than use the same <a> tag with your file's name: <a href="file-name.html">

If your HTML file is in a folder inside your folder, you need to add the folder's name to the hyperlink attribute:
<a href="folder-name/file-name.html">
