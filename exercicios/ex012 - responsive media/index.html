Implementing Videos

Local Host

To add videos to your website, you can host them locally by using the following tag:

<video src="file-name.mp4"> and add these attributes: controls, width, poster, autoplay, loop

This video is hosted in my own server

You can also use the <source> tag to include more format options for your users

External Host

To add a external hosted video from YouTube to your website, all you need to do is copy/paste the 'Embed' code in the 'Share' options to your HTML file
