Header, Main and Footer

To create a semantic website, is recommended to use Grouping tags such as <header>, <main> and <footer>

You can also use the <section>, <article> and <aside> tags (usually inside the main tag) to create a more readable site

Adding Shadows to your box

To add shadow to a box, you simply need to use the command 'box-shadow: right-leftpx top-bottompx blurpx spreadpx color';

Border Radius

To change the border radius of an element, you'll need to use the 'border-radius: px or %;
You can use that to create smooth looking boxes to your website

Border Image

To create a personalized border, you need to save the image you want in your folder and use the command: 

border: 10px solid trasparent;
border-image-source: url('folder-name/image-name.png');
border-image-slice: x;
border-image-repeat: repeat/stretch;

ShortHand:

border-image: url('folder-name/image-name.png') x repeat/stretch;
