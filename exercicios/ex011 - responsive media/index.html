Responsive Images

Using the tags <picture> and <source media="(min-width:)" srcset="your-img.png" type="image/png"> to create responsive images

Responsive Audio

Creating responsive audio with <audio> and <source src="file-name.mp3" type="audio/mp3"> tags

Inside the tag <audio>, you have different attributes like; 'preload="auto/metadata/none"', 'controls', 'loop', 'autoplay'
