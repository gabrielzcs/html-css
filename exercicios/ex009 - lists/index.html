Types of Lists

Ordered Lists

To create one you need to use the <ol> and <li> tags

  1. Wake up
  2. Go outside
  3. Eat breakfast
  4. Read my book
  5. Go to the gym
  6. Study HTML

You can change the type of list by adding the attribute "type": <ol type="1/A/a/I/i">

Unordered List

To create an Unordered List, just use the <ul> and <li> tags

  • Apple
  • Tomato
  • Onions
  • Carrot
  • Juice
  • Whey Protein

You can change the type of list by addign the attribute "type": <ul type="disc/circle/square">

Nested Lists

To create a Nested List, all you need to do is create a list inside your list. It can be either an Ordered List or an Unordered List

  1. Old games
    • Super Mario
    • Point Blank
  2. New games
    1. Counter-Strike:GO
    2. Valorant

Description Lists

To create a Discription List you need to use the <dl>,<dt> and <dd> tags
