Using External Fonts

If you want to download a font from a website (im using dafont), you'll need to use the @font-face rule to creat it, like this: 

@font-face{
  font-family: 'font-name';
  src: url('folder-name/font-name.otf') format('opentype'),
  url('folder-name/font-name.ttf') format('truetype');
}

You can also use the Fonts Ninja Google Chrome extension to get a font from any website

And also What Font Is, Font Squirrel and the My Fonts websites to find fonts from images
