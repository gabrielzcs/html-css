Paragraph and Line Breaks + Symbols

You can write however you want. You just need to use the tag <p> and </p>

But if you need to break a line.
like this, just use the tag <br>

& lt; to add a "Lower Than <" sign
& gt; to add a "Greater Than >" sign

To add a comment inside your code, type <!--Comment here-->

You can add symblos by using & commands:
® © ™ € £

And add emojis by using & #x"emoji code" (that you can get on the emojipedia.org):
🖖 🤓
