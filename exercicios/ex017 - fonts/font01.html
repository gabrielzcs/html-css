Fonts

Choosing a Font (font-family)

To choose a font you need to use the font-family property in your CSS file. This property can hold multiples font names as backup in case a specific browser doesn't support the one you chose first. Always put a generic name in the end like: sans-serif, serif, monospace, etc. To make sure that if none of the fonts you chose are compatible, it will use a deafult font of the browser

Metrics (font-size)

The recomendation is to use px(pixels) and em(size compared to the default browser font size. 1em = 16px)

Absolute Metrics:

cm, mm, in, px, pt, pc

Relative Metrics:

em, ex, rem, vw, vh, &

Font Weight (font-weight)

Font weight shows how thick of thin a font is gonna be

It can be: Lighter, Normal, Bold, Bolder

Shorthand (font:)

Shorthand is used to make it easier to configurate your font. It does that by making it one line command only, like this:

font: (font-style), (font-weight), (font-size), (font-family);

font: italic, bolde, 15px, 'Courier New', Courier, monospace;

You only need to type the names you want. In this exect order
