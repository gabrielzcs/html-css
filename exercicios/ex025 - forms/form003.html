Other types of Forms

Some other attributes you can also use are:

maxlength: it determines whats the lenght of what they can write

placeholder: let you write a small text to inform the user what that box is for

autocomplete: if you have a box for name, you can use the autocomplete="name" so that your browser knows what it is

max, min and step: these are for number. Max numbers, min numbers and steps is to set how many numbers it can skip

required: if you use this attribute, the user has to sign the box
