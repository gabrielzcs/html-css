Implementing ID's and Classes to CSS

ID's

So in the HTML file you'll use the id="id-name" to every tag you want different than the other. So for example:

You can have multiple paragraphs, but if you wanna alter only one of them, that's when you use the id: <p id="principal">

Now in your CSS file, you'll use the #id-name to style that especific id:

#principal or p#principal to be even more especific

And thats the result:

This paragraph can be on the right side of the screen

And this paragraph will stay on the left side

But id's can only be yoused once, you can't have that 'principal' id in multiple paragraphs. For that we use Classes

Classes

Classes work the same as id's but can be used in as many tags as you want:

class="class-name"

All you need to do is add this to the tag or tags you want different

And edit in CSS by using:

.class-name
